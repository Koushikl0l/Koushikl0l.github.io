Multimodal AQI Prediction Paper

Introduction

This paper explores a multimodal approach to predict the Air Quality Index (AQI) using numerical data and sky/air images, enabling image-only inference for mobile applications.

AQI Estimation on Image

Box 1: Data Collection and Preprocessing

Numerical data is normalized, and images are resized to 224x224 pixels with augmentations for robustness.

Box 2: Model Architecture

Features a numerical branch, image branch (EfficientNetB0), and a robust fusion layer with attention and contrastive learning.

Writing Board

Box 3: Training and Evaluation

Trained with PyTorch using AdamW, validated with RMSE and MAE, showing improved accuracy in diverse conditions.

Box 4: Deployment and Future Work

Deployable via TorchScript; future work includes temporal modeling and vision transformers.

Conclusion

The model effectively predicts AQI with a robust fusion mechanism, contributing to environmental monitoring.
